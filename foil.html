FOILing Basics

Here, we will cover the most basic FOILing case - 2-digit by 2-digit multiplication.

We have two 2-digit numbers to multiply - ab and cd where ab = (10×a+b) and cd = (10×c+d).

Then, ab × cd = (10×a+b) × (10×c+d) = 100×(a×c) + 10×(b×c+a×d) + 1×(b×d).

(b×d) represents the units digit, (b×c+a×d) represents the tens digit, and (a×c) represents the hundreds digit. If any of these values are greater than or equal to 10, then you carry over to the next higher tens.

Example: 43×89

Units = 3×9 = 27

Tens = 3×8 + 9×4 + 2 = 62

Hundreds = 4×8 + 6 = 38

= 3827
